# Hector-Bridge Splitter Contract

### Development

#### Setup

Install dependencies with [Yarn](https://yarnpkg.com/en/):

```bash
yarn install
```

#### Testing

```bash
yarn test
```

#### Deployment

Edit [hardhat.config.ts](hardhat.config.ts) and copy sample.env to .env and input the relevant variables, then run:

```bash
yarn deploy:[network]
```
