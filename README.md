# TKN Assessment

### Development

#### Setup

Install dependencies with [Yarn](https://yarnpkg.com/en/):

```bash
yarn install
```

#### Deploy

```bash
yarn deploy
```

#### Verify

```bash
yarn verify
```

#### Testing

```bash
yarn test
```
